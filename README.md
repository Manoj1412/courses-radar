# Syllabus Radar (CourseRadar)

## Quick Start

1. `npm install`
2. Add `.env`:
```
YOUTUBE_API_KEY=your_key
```
3. `npm run dev` (localhost:8080)

## Deploy Vercel

- Connect GitHub repo
- Add env var: `YOUTUBE_API_KEY`
- Build: `npm run build`
- Output: `dist`

## Features
- YouTube course search/ranking
- Syllabus match
- Filters, bookmarks
